Semáforo em Arduino

Link do Projeto

O projeto era construir um protótipo de sistema de semáforos sincronizados baseado no Arduino para um cruzamento com via primária de controle e via secundária, ambas com botões para pedestres. O sistema possui dois semáforos, cada um representado por um conjunto de 3 LEDs (vermelho, amarelo e verde). O tempo de ligado e desligado de cada LED foi programado de acordo com o diagrama feito pelo professor apresentado em sala.

Após feita a parte 1 no Arduino físico, tivemos que passar para o TinkerCad para possibilitar o uso detalhado do potenciômetro. A via principal sofre variação no fluxo de veículos, dependendo de fatores aleatórios imprevisíveis. Por outro lado, o fluxo na via secundária é constante. Por isso, optou-se pela instalação de um sensor na via principal (potenciômetro), possibilitando ajustar os tempos de verde de cada semáforo de forma a otimizar o fluxo global de veículos.

O circuito inclui dois botões para solicitação de travessia de pedestres, um para cada via. O acionamento de qualquer um dos botões deverá causar o respectivo tratamento, fazendo com que os dois semáforos de veículos fiquem no vermelho, enquanto os dois semáforos de pedestres fiquem no verde a fim de permitir a travessia. O acionamento do botão não causa o fechamento imediato para os veículos, é preciso aguardar ao menos o fim do ciclo corrente dos semáforos dos veículos, para então, fazer o tratamento efetivo do acionamento do botão.
