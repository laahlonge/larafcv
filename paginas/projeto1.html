Projetos em Arduino

De Mim Para Mim

Jogo 2D feito no Construct com cenas de animação. A ideia e história do jogo foi criada em grupo, nos dividimos na realização do jogo, e fui responsável pelo desenho da maior parte dos personagens, desenho e edição das animações

Trabalho acadêmico realizado como projeto inicial da disciplina de 1° Período Experiência Criativa: Navegando na Computação, ministrada pelos professores Luiz Antonio Pavão e José Eduardo Lino




Sistema Bancário em Python

Sistema bancário onde, no cenário do professor, minha equipe foi contratada para realizar a implementação de um sistema bancário em Python com duas interfaces: uma para o cliente e outra para o gerente. O sistema deveria ser executado via linha de comando e os dados seriam armazenados toda vez que o sistema fosse fechado

Trabalho acadêmico realizado como projeto em grupo para a disciplina de 1° Período Raciocínio Algorítmico, ministrada pelo professor Jean Paul Barddal








Modelagem de Banco de Dados

Projeto e implementação de um Banco de Dados atendendo aos padrões adotados na disciplina. Possui informações textualizando o cenário selecionado, relação de 22 informações a serem extraídas a partir do banco de dados, Diagrama Entidade Relacionamento (DER), todas as dependências funcionais, modelo lógico normalizado até a 3 FN, dicionário de variáveis e prova de conceito (POC), além do arquivo em SQL

Trabalho acadêmico realizado como projeto final para a disciplina de 3° Período Banco de Dados, ministrada pela professora Deborah R Carvalho








Exercício de cards HTML e CSS

Exercício de sala onde deveríamos criar cards de produtos em linha, começando da esquerda para a direita. Os cards deveriam conter foto, descrição e preço. Ao clicar em "Comprar", deveria mostrar na tela a descrição do produto comprado

Trabalho acadêmico realizado como exercício em sala para a disciplina eletiva de Programação Web, ministrada pelo professor José Eduardo Lino
