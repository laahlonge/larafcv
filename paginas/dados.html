Modelagem em Banco de Dados

Link do Projeto

O projeto consistia em criar um cenário para a realização do banco de dados e todos os processos envolvidos. Foi-se necessário criar o Diagrama Entidade Relacionamento (DER), uma relação das 22 informações a serem extraídas, todas as dependências funcionais normalizadas até a 3FN, o dicionário de variáveis e a Prova de Conceito (POC) detalhada das 20 instruções SELECT versus dados populados nas tabelas.

Uma certa rede de escolas é separada por unidade, cada uma com um nome único, localizada em uma cidade. Uma data determina quando a unidade foi fundada. Cada unidade possui blocos diferentes, cada um contendo uma faixa etária de alunos, e com inspetores responsáveis por uma área da unidade (bloco). Cada bloco possui diversas salas de aula com diversas turmas e alunos. Cada unidade escolar tem seus diferenciais, como de aulas diferentes, cada uma com um nome; além de professores e inspetores para cada unidade. Cada aluno participa de uma única turma, mas pode participar de várias aulas. Um professor pode estar vinculado a uma unidade apenas, mas pode trabalhar em diversos blocos, tendo uma carga horária composta por cada aula que leciona. Um inspetor pode estar vinculado a uma unidade apenas, mas poderá ser responsável por diferentes blocos, tendo uma carga horária determinada por quantas horas ele trabalha na unidade. Uma rede de transporte escolar disponibiliza diversos tipos de transporte para buscar e levar os alunos, dentre eles ônibus e vans, dependendo da quantidade de alunos a serem transportados em determinado bairro.
